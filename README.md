# HTML Text Formatting Tags

This project demonstrates different HTML text formatting tags using simple examples and styling.

## Features

* Bold Text
* Italic Text
* Underline Text
* Strong & Emphasized Text
* Subscript & Superscript
* Marked Text
* Inserted & Deleted Text
* Nested Formatting Tags
* Horizontal Rule

## Technologies Used

* HTML5
* Inline CSS

## Purpose

The purpose of this project is to understand and practice HTML text formatting tags and basic webpage styling.

## Output

The webpage displays examples of various formatting tags with colorful styling and formatted text.
